Prinipais formatações

Negrito / Destaque:

No HTML5 se utiliza a tag < STRONG > para deixar um paragrafo em negrito/destaque e não se utiliza mais a tag < B > para deixar o texto em negrito.

Itálico / Ênfase

No Html5 se utiliza a tag < EM > para deixar o texto em italico/ênfase e não se utiliza mais a tag < I > para deixar o texto em Itálico.

Texto marcado

Podemos criar um texto marcado usando a tag <MARK>

podemos mudar a cor do texto marcado com o CSS e aqui vamos utilizar o estilo/forma dentro da tag <mark>.

A maneira certa de criar um estilo/forma é abrindo o style dentro ou na pasta do seu html, aqui vamos utilizar dentro do html. Repare que o que estilo que foi criado dentro da tag <mark> não foi alterado,e sim apenas na tag mark pura.

Texto pequeno

Aqui um texto pequeno utiliando a tag <small>

Texto excluido/deletado

Podemos marcar um texto excluido/deletado para indicar que ele deve ser lido, mas não considerado. utilizando a tag < del >

Texto insirido

Podemo marcar um texto como inserido para dar uma ênfase e indicar sua importância, utilizanda tag < ins >

Texto sobrescrito

Para inserir texto elevado: x20 , utilizando a tag < sup >

Texto subscrito

Para inserir um texto subscrito: h20, utiliza-se a tag < sub >
